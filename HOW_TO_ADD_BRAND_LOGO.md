# How to Add Your Brand Logo

## Method 1: Manual Upload (Easiest)

### Step 1: Save Your Logo File
1. Save your logo image to your computer
2. Rename it to: `brand-logo.png` (or `.jpg`, `.svg`)
3. Copy the file to: `C:\wamp64\www\tailoring\uploads\logos\brand-logo.png`

### Step 2: Refresh Your Browser
1. Press **Ctrl + F5** to hard refresh
2. âœ… Your logo will appear automatically on:
   - Login page
   - Registration page  
   - Sidebar menu
   - All pages

---

## Method 2: Via Company Settings

1. Login to your account
2. Go to **Company Settings**
3. Click on **"Business Logo"** section
4. Click **"Choose File"**
5. Upload your logo
6. Save

This will use your logo for invoices and company branding.

## Troubleshooting

**Logo not showing?**
- Check file name is exactly: `brand-logo.png`
- Check file is in: `uploads/logos/` folder
- Clear browser cache (Ctrl + F5)
- Verify file permissions (readable)

**Logo looks blurry?**
- Use higher resolution image
- Try PNG or SVG format
- Increase image quality

**Logo doesn't fit?**
- Resize to recommended dimensions
- Use landscape orientation
- Maximum height: 80px for best fit
